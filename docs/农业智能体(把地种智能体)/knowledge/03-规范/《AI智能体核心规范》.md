# 2. AI核心层总体定义

---

# 2.1 AI核心层定位

AI核心层不是：

```text
简单聊天机器人

```

也不是：

```text
普通问答系统

```

AI核心层定义为：

> 面向农业生产任务的智能决策与执行中枢。

核心职责：

*   理解农业任务
    
*   分析环境状态
    
*   调用农业知识
    
*   进行农艺推理
    
*   生成农业处方
    
*   判断风险
    
*   调度设备能力
    
*   控制执行流程
    
*   形成任务闭环
    

---

# 2.2 AI核心层目标

系统需实现：

| 能力 | 说明 |
| --- | --- |
| 任务理解 | 理解自然语言任务 |
| 多模态感知 | 识别文本、图像、环境数据 |
| 农业推理 | 基于农业知识分析 |
| 能力调度 | 判断设备是否可执行 |
| 风险控制 | 判断是否允许执行 |
| 智能编排 | 自动生成执行流程 |
| 状态管理 | 保持任务上下文 |
| 反馈闭环 | 根据结果优化任务 |

# 5. 意图控制系统（Intent Control）

---

# 5.1 模块目标

控制：

```text
用户到底是在“查看”还是“执行”

```

防止：

```text
一句自然语言
直接启动农机

```
---

# 5.2 意图分类

| 类型 | 示例 | 是否允许自动执行 |
| --- | --- | --- |
| VIEW | 看看葡萄熟了吗 | 否 |
| ANALYSIS | 是否缺肥 | 否 |
| ADVISORY | 最近做什么农事 | 否 |
| CONDITIONAL\_EXECUTE | 如果有病害就打药 | 可执行 |
| DIRECT\_EXECUTE | 直接收割 | 需确认 |
| HIGH\_RISK | 大面积喷药 | 强制确认 |

---

# 5.3 意图结构

```json
{
  "intent_type": "CONDITIONAL_EXECUTE",
  "target": "grape",
  "action": "spray",
  "auto_execute": true
}

```
---

# 5.4 意图控制规则

系统必须遵循：

```text
分析类任务
禁止自动执行农机

```

只有满足：

```text
意图允许
∧ 风险允许
∧ 能力允许

```

才允许执行。

# 10. 能力模型系统（Capability Model）

---

# 10.1 模块目标

统一描述：

```text
设备能做什么

```
---

# 10.2 能力模型定义

能力不是设备名称。

能力是：

```text
设备具备的动作集合

```
---

# 10.3 标准能力结构

```json
{
  "device_id": "robot_001",
  "capabilities": [
    "move",
    "pick",
    "spray"
  ]
}

```
---

# 10.4 能力分类

| 能力 | 说明 |
| --- | --- |
| move | 行驶 |
| pick | 采摘 |
| spray | 喷药 |
| sow | 播种 |
| fertilize | 施肥 |
| harvest | 收割 |
| transport | 运输 |

---

# 10.5 能力匹配原则

任务只能由：

```text
具备对应能力的设备执行

```

例如：

```text
任务需要 spray
设备必须具备 move + spray

```

```text

```
---

# 11.2 风险分类

| 风险类型 | 示例 |
| --- | --- |
| 环境风险 | 大风、大雨 |

# 11.3 风险输出

```json
  }

```

# 11.4 风险门控规则

系统必须执行：


```text
AI绕过风险系统直接执行

```

# 14. 智能体编排系统

---

# 14.2 编排流程

```text

→ 意图识别
→ 农情分析智能体

→ 农机执行智能体

```
---

# 14.3 多智能体协同原则

各智能体：

*   只能负责自己的职责
    
*   不允许跨边界控制
    
*   必须通过统一任务流协作

# 20. 最终定义

AI核心层定义为：

> Agri-Agent OS 中负责农业任务理解、农业推理、风险控制、能力调度与任务编排的智能决策中枢。

AI核心层不是：

```text
农机控制器

```

而是：

```text
农业智能决策系统

```

其核心目标是：

> 让农业生产从“人控制机器”演进为“AI理解任务并调度农业能力”。
